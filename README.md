# Playwright API Framework

A REST API testing framework built with Playwright.

## Tech Stack
- Playwright
- TypeScript / JavaScript
- Node.js

## Getting Started

### Install dependencies
```bash
npm install
```

### Run tests
```bash
npx playwright test
```

## Project Structure
```
├── tests/          # Test files
├── utils/          # Helper functions
├── config/         # Configuration files
└── .gitignore
```
